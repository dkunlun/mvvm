# mvvm
基于数据劫持的数据双向绑定的实现

## git clone

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build
```

## 目录结构
```
|
|——src 资源文件
| |—— js
| | |—— observer 数据劫持
| | |—— watcher 监听器
| | |—— compile  编译器
| | |—— mvvm 入口js
| |——————————————————
|—— index.html 入口html
|________________________________________________

```
